# Shikimori Temset 2 (Modern Glass & Twitch style)

Современная тема для профиля Shikimori со стеклянными элементами (Glassmorphism), плавной анимацией и акцентами в стиле Twitch.tv.

## Установка

1. Перейдите в настройки профиля на Shikimori: [Настройки -> Стили сайта](https://shikimori.io/profiles/edit/styles).
2. Задайте Вашу фоновую картинку в поле **"Фоновая картинка"** (дизайн автоматически применит к ней эффект прозрачного стекла).
3. В поле **"Стили сайта"** вставьте следующий код:

```css
@import url('https://raw.githubusercontent.com/S0S1S0CHKA/shikimori-temset2/main/styleV3.css');

:root {
  /* Здесь вы можете переопределить основной цвет, если стандартный фиолетовый не подходит */
  /* --accent-primary: #9146FF; */
  /* --accent-hover: #772CE8; */

  /* Настройки прозрачности стекла (по умолчанию 0.65) */
  /* --glass-bg: rgba(18, 18, 20, 0.65); */
}
```

4. Сохраните изменения.

## Особенности

- **Динамический фон**: все виджеты просвечивают фон сайта с эффектом размытия (Apple Glassmorphism), подстраивая оттенки под Ваш фон.
- **Twitch accents**: плавные наведения, свечение кнопок и ссылок.
- **Интерактивность**: 3D эффекты при наведении на обложки (карточки аниме/манги, постеры).
- **Минималистичный код**: Вам не нужно вставлять кучу кода в `Стили сайта`, достаточно одного импорта.
